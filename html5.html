Parrafos para argumentar ideas o textos

Las etiquetas de apertura y cierre por ejemplo div, section, header, parrafos, h1,h2,h3... Son etiquetas con un princpio y un final, etiquetas que se abren y se cierran.

Se usa para definir estructuras y contenido de paginas web

Las etiquetas de html5 estan definidas de muchas maneras, tenemos etiquetas "generales" ya definidas con su propio nombre, por ejemplo "header", "body", "footer" y entre muchas mas que podemos encontrar. Y por otro lado podemos encontrar "div" "li" "lu" "nav" entre muchas que hay.
